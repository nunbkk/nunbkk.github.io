# Project Avian Rebellion — MVP v8 (สรุปฉบับปัจจุบัน)

Tower defense 2D แนว LINE Ranger ธีมกองทัพนกปะทะกองทัพหุ่นยนต์แมว
ไฟล์เดียวจบ (`HTML + CSS + JS`) รันในเบราว์เซอร์ ไม่ต้องมีเซิร์ฟเวอร์ ไม่มีการซื้อด้วยเงินจริง

## 4. ระบบธาตุ (เฉพาะ MVP slice นี้)

วงจร 4 ธาตุ: **ดิน → ลม → น้ำ → ไฟ → (วนกลับดิน)**
ธาตุที่อยู่ก่อนหน้าในวงจร "ชนะ" ธาตุถัดไป → โจมตี ×1.5 (คริติคอล) / โดนสวนกลับ ×0.67
ธาตุ **ไม้** (บอสทุกตัว) เป็นกลาง ไม่แพ้ไม่ชนะธาตุใดในระบบนี้

## 7. ระบบคลื่นมอนสเตอร์ (Wave System)

- แต่ละด่านมี 3 เวฟ เวฟสุดท้ายของด่าน 5/10 คือบอส
- **เคลียร์เวฟหมด → เวฟถัดไปออกทันที** (0.0 วินาที)
- **ถ้าเวฟค้างนานเกินเกณฑ์** เวฟถัดไปเริ่มทยอยออกซ้อนทับเลย ไม่ต้องรอฆ่าหมด:
  - ด่าน 1–2: เกณฑ์ **10 วินาที**
  - ด่าน 3 ขึ้นไป: เกณฑ์ **15 วินาที**
- **มอนสเตอร์ทยอยออกทีละตัว** (บางครั้งสุ่มออก 2 ตัวพร้อมกัน) ทุกๆ 0.7–1.5 วินาที ไม่ออกมารวดเดียวทั้งเวฟ ดูเป็นธรรมชาติ

## 8. ระบบการต่อสู้

- **หน่วยมีลี**: เข้าประชิดระยะ 7% ของสนามแล้วฟันทันที (มีอนิเมชันพุ่งเข้าหา)
- **หน่วยระยะไกล** (โอเชี่ยน ฝั่งเรา / Striker ฝั่งศัตรู): หยุดยิงจากระยะ **20%** ของสนาม (~3 ตัวละคร) ไม่กรูเข้าประชิด มีอนิเมชันสะบัดตัวถอยหลังตอนยิง (recoil) แทนการพุ่งเข้าหา
- **กระสุนบินจริง**: ดาเมจคำนวณตอนยิง แต่ **จะเข้าจริงตอนกระสุนบินถึงเป้าหมาย** (หน่วง 110–260ms ตามระยะทาง) ถ้าเป้าหมายตายไปก่อนกระสุนถึง จะไม่เกิดผล (เหมือนยิงพลาด)
- มอนสเตอร์ทั่วไปที่เดินไปถึงป้อมนก จะปักหลักทุบป้อมต่อเนื่องเหมือนบอส (ไม่หายตัว) — สร้างแรงกดดันของจริง

## 9. อินเทอร์เฟซและ Feedback

- **แบนเนอร์กลางจอ**: ขึ้นตอนเข้าด่าน (ชื่อด่าน+World), ตอนบอสปรากฏตัว, ตอนใช้สกิล (มี throttle 2s/ชนิด กันสแปมเมื่อนกชนิดเดียวกันหลายตัวปล่อยสกิลไล่กัน), ตอนหมดเวลา — วางตำแหน่งใต้แถบเลือดบอส ไม่บังตัวละคร
- **นาฬิกานับถอยหลัง** มุมกลางบนจอ (⏱ 2:00 → 0:00) เตือนสีแดง+เต้นเมื่อเหลือ ≤20 วิ
- **ปุ่มสกิล**: เปลี่ยนเป็นตัวแสดงสถานะอัตโนมัติ (auto) ไม่ต้องกด แสดงคูลดาวน์ของตัวที่ใกล้พร้อมที่สุด + จำนวนตัวในสนาม (เช่น "×2")
- **ผลจบด่าน (win/lose)**: แสดงดาว, ดาเมจรวม, ศัตรูที่กำจัด, และ**สถิติดาเมจแยกตามนกแต่ละตัว เรียงจากมากไปน้อย** พร้อมแถบเทียบสัดส่วนสี
- **วิกเน็ตแดงขอบจอ** เตือนเมื่อป้อมนก HP <20% (จังหวะหายใจช้า ไม่กระพริบถี่แสบตา)
- **จอสั่น**: สั่นแรงเฉพาะตอนเราคริติคอลเอง/บอสใช้สกิล/แพ้ ส่วนบอสทุบป้อมประจำใช้สั่นเบาๆ แทน (ลดความรำคาญ)

## 10. ภาพและฉาก

- **ป้อมทั้งสองฝั่ง** ออกแบบใหม่เป็น SVG วางชิดขอบจอพอดี:
  - ฝั่งนก (ซ้ายสุด): บ้านต้นไม้ มีรัง ธง บันไดเชือก หน้าต่างไฟอุ่น
  - ฝั่งศัตรู (ขวาสุด): หอคอยยอดหูแมว หน้าต่างตาแมวเรืองแสง เสาอากาศไฟแดงกะพริบ
- **ฉากหลังสุ่มใหม่ทุกครั้งที่เข้าด่าน**: ต้นไม้ ก้อนหิน กอหญ้า ท่อนไม้ เศษฟาง ตำแหน่ง/ขนาด/สี สุ่มหมด
- **นกจางบินผ่านท้องฟ้า** (ทุกด่านตั้งแต่ด่าน 1): ขนาดสุ่ม 22–50px, สุ่มทิศซ้าย↔ขวา, ความสูง, ความเร็ว, ช่วงเวลาออกสุ่ม 3–10 วินาที บางครั้งบินมาเป็นคู่
- **ตัวละคร/มอนสเตอร์**: นก 76px, มอนสเตอร์ทั่วไป 70px, บอส 150px — มีเงาใต้ตัว, ขยับตัวเบาๆ ตอนยืน (idle bob), เอฟเฟกต์จางหายตอนตาย
- **เอฟเฟกต์สกิลแยกกันชัดเจนทุกตัว**: โจโจ้ = โดมโล่ฟ้า, ข้าวปั้น = เปลวไฟ+ลูกไฟลอย, โอเชี่ยน = คลื่นน้ำวงแหวนซ้อน, โมโม่ = เสาแสงเขียว+หัวใจ/ดอกไม้ลอย, บอส = กรงเล็บแดงฟาด 3 เส้น

## 11. เสียง

- **สร้างด้วย WebAudio ล้วนๆ** ไม่มีไฟล์เสียงภายนอก
- **แยกโทนเสียงฝั่งเรา vs ฝั่งมอนสเตอร์อย่างชัดเจน**: ฝั่งเราใช้ oscillator tone สดใส (sine/triangle) ฝั่งศัตรูใช้ white-noise ผ่าน filter ให้เสียงกร้าน/หยาบ (growl/hiss)
- **สกิลแต่ละตัวมีเสียงเฉพาะ** ไม่ซ้ำกัน
- **BGM ประจำด่าน**: สุ่มทำนอง/คีย์/จังหวะจาก seed เลขด่าน (เข้าด่านเดิมได้เพลงเดิม ด่านต่างกันได้เพลงต่างกัน) ด่านปกติ = คีย์เมเจอร์สดใส, ด่าน MiniBoss/WorldBoss = คีย์ไมเนอร์ 168 BPM มีกลอง/ไฮแฮทตื่นเต้น และสลับเป็นโหมดบอสอัตโนมัติเมื่อบอสปรากฏกลางด่าน

## 13. ไฟล์และเทคนิค

- ไฟล์เดียว `.html` รวม CSS/JS/SVG ทั้งหมด ไม่มี dependency ภายนอกอื่นนอกจากฟอนต์
- ยูนิตใช้ **persistent DOM element + diff ต่อเฟรม** (ไม่ทุบสร้างใหม่ทุกเฟรม) เพื่อลด flicker และภาระเครื่อง
- บันทึกความคืบหน้า (ด่านที่ปลดล็อก, ดาวที่ได้) ผ่าน `localStorage`
